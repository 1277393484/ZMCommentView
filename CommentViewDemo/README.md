# ZMCommentView
1、可扩展、可单独分离使用的弹出式评论列表，一般用于查看全部评论；

2、键盘弹出评论输入，自适应文字高度；

3、仅引入了Masonry做布局；

4、一句代码展示全部评论

有什么问题，可以再issues告知我，欢迎点小星星！


使用方式：

 [[ZMCusCommentManager shareManager] showCommentWithSourceId:nil];
